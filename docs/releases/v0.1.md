# v0.1 小程序测试版闭环

## Purpose
让用户可以在微信小程序中完整体验「要过期」的核心闭环：进入小程序、设置昵称、创建或加入家庭、切换家庭、录入物品、查看临期/过期状态、编辑、标记用完和删除。

## Included Scope
- 微信云开发真实联调。
- 微信登录身份接入，云函数可获取 `OPENID`。
- 手动设置昵称。
- 创建家庭。
- 切换家庭。
- 加入家庭的最小可用流程。
- 新增、编辑、删除物品。
- 标记物品已用完。
- 多图上传、首页封面和详情预览。
- 首页搜索、状态统计、临期/过期展示。
- 下拉刷新、详情返回首页刷新、编辑/新增后定位目标物品。
- 基础空状态、错误提示和加载反馈。

## Excluded Scope
- 真实微信订阅消息推送。
- Android/iOS 正式 App 适配。
- 复杂家庭角色和权限细分。
- 大数据量分页、虚拟列表和后端游标分页。
- OCR、条码识别、AI 建议。
- 上架发布、隐私协议和生产运营配置。

## Required Quality Bar
- 微信开发者工具中可完成完整主流程。
- `pnpm typecheck` 通过。
- `pnpm test` 通过。
- `pnpm build:mp-weixin` 通过。
- 云函数返回结构与 `docs/api.md` 的 `validated` contract 一致。
- 页面仍通过 service/repository/adapter 访问数据，不直接依赖 mock 文件或 cloud client。

## Exit Criteria
- [ ] TASK-011 完成：真实云开发联调入口和主流程跑通。
- [ ] TASK-012 完成：家庭切换可用。
- [ ] TASK-013 完成：图片云存储 adapter 正式接入。
- [ ] TASK-014 完成：加入家庭/邀请最小流程可用。
- [ ] TASK-015 完成：v0.1 测试版验收与体验修复。
- [ ] 用户确认 v0.1 测试版可以作为内部体验版本。
