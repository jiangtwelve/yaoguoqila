# Decision 005: 过期日期录入方式

## Context
不同物品包装上的日期信息不同。有些物品直接标注到期日，有些物品标注生产日期和保质期。单一输入方式会让一部分物品录入变得别扭。

## Decision
新增/编辑物品表单同时支持两种日期录入方式：
- 直接填写过期日期。
- 填写生产日期 + 保质期，自动计算过期日期。

两种方式在表单中用「或」分隔，用户自由选择。

## Consequences
- 数据模型需要记录 `expiryInputMode` 以便编辑页回显。
- 无论使用哪种录入方式，系统都生成统一的 `expiresAt`，用于首页排序、状态计算和提醒。
- 原型需要重点验证表单不会因为两套日期输入变得拥挤或难懂。
